# C++ Quiz System
A simple console-based quiz system developed in C++ using file handling and a timer.

## Features
- Multiple choice questions
- Score calculation
- Leaderboard system
- File handling using fstream
- Timer for each question

## Files
- QuizSystem.cpp -- Main program logic
- questions.txt -- Stores quiz questions
- leaderboard.txt-- Stores user scores

## How to Run
1. Compile using a C++ compiler :  g++ QuizSystem.cpp -o quiz
2. Run : ./quiz
